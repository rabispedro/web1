Exercício 1

Manipulação de elementos JavaScript:

Escrever um código de uma função que verifique se o campo nome de uma pessoa está vazio (campo requerido/obrigatório) por meio de JavaScript. Apresente mensagens de alerta.

Escrever um código de uma função que faça a validação de um campo CPF de uma pessoa em um formulário. O campo deve ser obrigatório e numérico com 11 dígitos. Apresente mensagens de alerta.

Exercício 2

Crie um elemento do tipo input e um do tipo span. O input deve ser do tipo texto

Faça uma função em JavaScript que apresente no elemento do tipo span quantos caracteres foram digitados a cada vez que uma tecla for solta. Por padrão, o lemento span deve apresentar o valor 0, quando o input estiver vazio.

Exercício 3

Crie um elemento do tipo input text. Além disso, inclua um botão com o label "Gerar".

Faça um script que leia o texto do input. Caso tenha menos de duas palavras, emita um alerta informando que deve ser informado um número superior de palavas. Caso exista pelo menos três palavras, o script deve quebrar toda a frase usando como separador o espaço em branco. Cada palavra será usada para compor um novo input com a edição desabilitada.

Exercício 4

Crie um formulário em HTML com os seguintes campos:

  • Nome (O campo nome não pode ser vazio, não pode conter apenas espaços, não pode ter mais de 40 caracteres)
  • Data de Nascimento (O campo data de nascimento não pode ser vazio. O ano de nascimento não pode ser inferor a 1910 e nem superior ao ano atual)
  • Quantidade Filhos (O campo quantidade filhos deve aceitar apenas "números" entre 1 e 99)
  • Botão enviar (utilize um input type submit)

Observação: não utilize atributos HTML para realizar a limitação e/ou validação dos campos solicitados. Faça através de script quando clicar no botão Enviar.
