Lista de Termos - tags dl, dt e dd

HTML
(abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web.
CSS
Cascading Style Sheet (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.
JavaScript
Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. JavaScript permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos da web. A grande maioria dos sites usa, e todos os principais navegadores têm um mecanismo JavaScript dedicado para executá-lo.
