Exercício 1

Faça o que se pede:

  1. Crie um elemento do tipo ComboBox (select) que conterá as opções: Escolha um estado; São Paulo; Minas Gerais; Rio de Janeiro e Espirito Santo
  2. Crie um novo ComboBox que conterá a opção: Escolha uma Cidade. Este elemento deve receber, inicialmente, o estilo display: none para que não seja exibido na página.
  3. Crie uma função que vai ser disparada no evento onchange do primeiro ComboBox criado (estados). Caso a opção escolhida seja diferente da correspondente a "Escolha um estado", o ComboBox criado no item b deve ser apresentado. Caso contrário, ele deve ser ocultado.

Exercício 2

Partindo do mesmo princípio adotado on exercício 1, isto é, criar um ComboBox para selecionar um estado e um ComboBox para Selecionar uma cidade, considere que agora, de acordo com o estado escolhido, deve ser apresentado no segundo ComboBox (de cidades) uma lista de cidades (pelo menos três) referente a cada estado.
